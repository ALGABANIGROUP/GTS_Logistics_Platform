## 🚀 EN | Getting Started

### 1. EN | Install Requirements
```bash
# Backend
cd backend
pip install -r requirements.txt

# Frontend
cd frontend
npm install
```

### 2. EN | Configure Environment
```bash
# Copy environment file
cp backend/.env.example backend/.env

# Edit configuration
nano backend/.env
```

### 3. EN | Run Database Migrations
```bash
cd backend
alembic upgrade head
```

### 4. EN | Start Servers
```bash
# Backend (Terminal 1)
uvicorn backend.main:app --reload

# Frontend (Terminal 2)
cd frontend
npm run dev
```

### 5. EN | Run Tests
```bash
# Unit tests
pytest backend/tests/ -v

# Integration tests
python scripts/run_integration_tests.py
```
