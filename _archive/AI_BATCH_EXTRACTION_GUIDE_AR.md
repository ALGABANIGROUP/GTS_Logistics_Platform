# 🚀 AI Invoice Extraction - Quick Guide

## ✨ New Features

### 1. All File Types Supported
✅ PDF  
✅ Excel (.xls, .xlsx)  
✅ Word (.doc, .docx)  
✅ Images (.png, .jpg)

### 2. Batch Processing
- Select up to **30 files** at once
- Separate extraction for each invoice
- Detailed results

---

## 🎯 How to Use

### Step 1: Open Expenses
```
Admin Panel → Platform Expenses → + Add Expense
```

### Step 2: Select Files
1. In the purple card (🤖):
   - Click **"Select Invoices (max 30)"**
2. Select multiple files:
   - Hold `Ctrl` + Click each file
   - Or drag to select multiple files
3. Select up to 30 files of any type

### Step 3: Wait for Extraction
- Takes ~3-5 seconds per file
- Progress bar shows status
- Do not close the page!

### Step 4: Save Data

#### If you selected one file:
- The form auto-fills
- Review the data
- Click "Save"

#### If you selected multiple files:
- Extracted data appears in Console (press F12)
- Save each invoice manually

## 💡 Important Tips

### ✅ Do:
- Use clear files
- Name files with understandable names
- Try one file first
- Make sure invoices are complete

### ❌ Do not:
- Do not select more than 30 files
- Do not use corrupted files
- Do not close the page during processing

---

## 🔧 Troubleshooting

### Problem: "Failed to extract"
**Solution**:
- Make sure the file quality is good
- Try a different file type
- Ensure there is clear text

### Problem: Inaccurate data
**Solution**:
- Review the data before saving
- Edit incorrect fields
- Use clearer images next time
