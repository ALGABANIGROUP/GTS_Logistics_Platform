## 📖 Usage Examples

### Access the Panel
```javascript
// Navigation
window.location.href = 'http://localhost:5173/freight-broker'

// or in React
<Link to="/freight-broker">Freight Broker</Link>
```

### Monitor Transport
```javascript
// Check real-time updates
WebSocket → /api/v1/transport/ws/tracking
// Updates: Truck locations, Shipment status, Routes
```

### Monitor Safety
```javascript
// Check safety alerts
WebSocket → /api/v1/safety/ws/alerts
// Updates: Incidents, Alerts, Metrics
```

### Control Bots
```javascript
// Start bot
POST /api/v1/ai/bots/freight_broker/run

// Check status
GET /api/v1/ai/bots/freight_broker/status

// Health check
POST /api/v1/ai/bots/freight_broker/health_check
```

### For Frontend

3. Implement virtual scrolling for alerts
4. Debounce WebSocket messages
5. Cache static data

3. Implement connection pooling
4. Batch process alerts
