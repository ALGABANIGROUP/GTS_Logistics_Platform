# Figure 4

## Title
Incident Detection and Response Flowchart

## Drawing Scope
Telemetry-driven detection, classification, and response workflow for logistics incidents.

## Caption
Figure 4 is a flowchart illustrating an incident detection and response process comprising ingestion of telemetry or external event data, classification of an operational event by incident type and severity, generation of one or more automated responses, and route or shipment-impact updates. Automated responses may include notifications, ETA adjustments, escalation, and alternative route generation based on the incident context.

## Elements Shown
- Telemetry ingestion
- External event inputs
- Incident classification
- Severity evaluation
- Automated response logic
- ETA update
- Route adjustment
- Escalation output
