# Figure 3

## Title
Payment Processing Architecture

## Drawing Scope
Unified multi-gateway payment architecture with currency normalization and webhook reconciliation.

## Caption
Figure 3 depicts a payment processing architecture comprising a gateway abstraction layer that exposes a common application interface to multiple payment providers including Stripe, Wise, and SUDAPAY. The architecture further includes multi-currency handling, a unified payment record store, exchange-rate support, and webhook or callback processing for asynchronous payment confirmation, refund handling, and reconciliation.

## Elements Shown
- Application payment interface
- Gateway abstraction layer
- Stripe integration
- Wise integration
- SUDAPAY integration
- Multi-currency conversion
- Unified transaction history
- Webhook processing
