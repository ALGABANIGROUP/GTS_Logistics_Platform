# Figure 2

## Title
Freight Matching Workflow

## Drawing Scope
End-to-end flow for intelligent freight matching and rate prediction.

## Caption
Figure 2 is a workflow diagram showing a freight matching process comprising load ingestion from one or more sources, load enrichment with geographic and route data, carrier scoring using operational criteria, predictive rate analysis using historical and current market inputs, and match selection or automatic assignment. In some embodiments, a human review path is optionally inserted before final carrier booking.

## Elements Shown
- Load ingestion
- Load enrichment
- Carrier scoring
- Predictive rate analysis
- Match ranking
- Optional human review
- Load booking
