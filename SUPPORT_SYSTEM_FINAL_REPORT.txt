🚀 DEPLOYMENT STEPS
═══════════════════════════════════════════════════════════════════════

STEP 1: Database Migration
───────────────────────────────────────────────────────────────────────
cd backend
python -m alembic -c alembic.ini upgrade head

Expected output:
  INFO [alembic.runtime.migration] Context impl PostgreSQLImpl.
  INFO [alembic.runtime.migration] Will assume transactional DDL.
  INFO [alembic.runtime.migration] Running upgrade  -> 550e8400, support_system

Verify with:
  psql $DATABASE_URL -c "\dt support*"
  
Output should show:
  sla_levels
  support_agents
  support_tickets
  ticket_comments
  ticket_activities
  ticket_history
  ticket_attachments
  knowledge_base_articles
  support_feedback
  support_statistics
  support_email_templates
  support_emails

STEP 2: Environment Configuration
───────────────────────────────────────────────────────────────────────
Update .env with:

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
SMTP_FROM_EMAIL=support@yourdomain.com
SMTP_FROM_NAME="GTS Support"

IMAP_HOST=imap.gmail.com
IMAP_PORT=993
IMAP_USER=your-email@gmail.com
IMAP_PASSWORD=your-app-password
IMAP_FOLDER=INBOX

SUPPORT_EMAIL=support@yourdomain.com
SUPPORT_PHONE=+1-800-SUPPORT
FRONTEND_URL=http://localhost:5173

For Gmail:
  1. Enable 2FA
  2. Generate app password
  3. Use app password (16 characters, no spaces)

STEP 3: Verify Routes Registered
───────────────────────────────────────────────────────────────────────
Check backend/main.py contains:

from backend.routes.support_routes import router as support_router
if support_router:
    app.include_router(support_router, prefix="/api/v1/support")

STEP 4: Start Backend
───────────────────────────────────────────────────────────────────────
.\run-dev.ps1

Expected logs:
  [main] support_routes mounted at /api/v1/support
  [startup] bot_os started
  [db] database connected

STEP 5: Start Frontend
───────────────────────────────────────────────────────────────────────
cd frontend
npm run dev

Expected output:
  ➜  Local:   http://localhost:5173/
  ➜  press h to show help

STEP 6: Test the System
───────────────────────────────────────────────────────────────────────

  1. Open http://localhost:5173/support/tickets
  2. Click "Create Ticket"
  3. Fill in details
  4. Submit
  5. Verify ticket appears in list

Agent Flow:

Admin Flow:
